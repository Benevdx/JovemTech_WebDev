Atividade 1 — Site de Restaurante

Construção de um site multi-página com HTML semântico, CSS e JavaScript puro

📋 Contexto da Atividade
Você foi contratado para criar o site de um restaurante fictício. O site deve ter três páginas interligadas por um menu de navegação, compartilhando o mesmo arquivo de estilos. O tema, nome, cores e conteúdo são de livre escolha — seja criativo!
📌 Instruções Gerais
  • 1 Utilize apenas HTML, CSS e JavaScript puro — sem frameworks, bibliotecas externas ou templates prontos.
  • 2 O CSS deve ser escrito dentro da tag <style> ou em um arquivo externo styles.css linkado em todas as páginas.
  • 3 O JavaScript deve ser escrito dentro da tag <script> na página que o utiliza.
  • 4 Cores, fontes e estilos são de livre escolha — o visual deve ser coerente entre as três páginas.
  • 5 O site deve apresentar conteúdo coerente e funcional com o tema escolhido.
  • 6 O código deve ter comentários explicando as partes principais — tanto no HTML quanto no CSS.
  • 7 A navegação entre as páginas deve funcionar corretamente ao clicar nos links do menu.
⚠️ Os arquivos de referência (exemplo.html, guia-html.html, guia-css.html) são materiais de apoio para consulta. Copiar o conteúdo deles diretamente sem adaptação será considerado plágio.

🗂️ Estrutura das Páginas
1

Página Inicial

Apresentação do restaurante, slogan, menu de navegação, seção de boas-vindas, cardápio resumido (3 pratos), seção "Sobre Nós" e formulário de contato.

index.html
2

Cardápio Completo

Página dedicada ao menu do restaurante, organizada por categorias (entradas, pratos principais, sobremesas). Cada item deve ter nome, descrição e preço.

cardapio.html
3

Reservas

Página com formulário de reserva de mesa (nome, data, horário, número de pessoas, observações) e informações de funcionamento do restaurante.

reservas.html
✅ Conteúdo Mínimo Exigido por Página
index.html
  • Menu de navegação com links para as 3 páginas
  • Seção de boas-vindas (h1 + p)
  • 3 cards de pratos em destaque
  • Seção "Sobre Nós" (2 parágrafos)
  • Formulário de contato
  • Rodapé com copyright
cardapio.html
  • Mesmo menu de navegação
  • Título da página (h1)
  • Mínimo 3 categorias (section)
  • Mínimo 2 pratos por categoria
  • Nome, descrição e preço em cada prato
  • Mesmo rodapé
reservas.html
  • Mesmo menu de navegação
  • Título da página (h1)
  • Formulário com mínimo 5 campos
  • Campos: nome, data, hora, pessoas, obs.
  • Informações de funcionamento
  • Mesmo rodapé
