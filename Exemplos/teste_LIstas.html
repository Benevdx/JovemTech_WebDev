Este é um e exemplo de página com listas e tabelas.

Lista Não Ordenada (ul)

Usada quando a ordem dos itens não importa:

  • Maçã
  • Banana
  • Laranja

Lista Ordenada (ol)

Usada quando a ordem é importante:

  1. Acordar
  2. Tomar café
  3. Estudar

Lista Ordenada com atributos

  1. Item C
  2. Item D
  3. Item E
  4. Item F

Lista de Definição (dl)

Usada para termos e descrições:

HTML
Linguagem de marcação para criação de páginas web
Permite estruturar conteúdo com elementos semânticos
CSS
Responsável pelo estilo das páginas
JavaScript
Linguagem de programação para interatividade
